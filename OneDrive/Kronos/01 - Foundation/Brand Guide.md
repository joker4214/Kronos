## 2. Visual identity

**Rebranded 2026-07-04.** Full replacement of the previous navy/gold/cream identity below — this is now the canonical palette/type system for anything new. Old identity kept in the changelog at the bottom of this section for reference only; don't design against it going forward.

### Colors (exact hex — never approximate)

| Token | Hex | Use |
|---|---|---|
| `--charcoal` | `#16181C` | Primary dark — backgrounds, headings, footer |
| `--charcoal-soft` | `#22252B` | Secondary dark surface |
| `--accent` | `#2F5FFF` | Single bold accent — CTAs, links, highlights, badges. This is the only color besides neutrals — don't introduce a second accent hue. |
| `--accent-light` | `#5C82FF` | Hover states, secondary accent text |
| `--white` | `#FFFFFF` | Base light background |
| `--gray-100` | `#F5F5F6` | Light section backgrounds |
| `--gray-400` | `#8A8F98` | Muted body text on dark backgrounds |
| `--gray-700` | `#3F434B` | Body text on light backgrounds |

This is a **clean, crisp, neutral-led** palette — white/charcoal/gray base, one electric-blue accent doing all the color work. Not warm/earthy, not multi-color. If a generated asset introduces a second accent hue (gold, green, purple, etc.) or leans soft/pastel, it's off-brand.

### Typography

- **Headings / display:** a crisp geometric sans (Space Grotesk or Sora-equivalent) — weights 500/600/700. No serif, no italic-for-emphasis; use the accent color for emphasized words instead (e.g. "recognizable <span style='color:var(--accent)'>brands</span>").
- **Body / UI:** Inter (sans) — weights 400–700 (unchanged)
- **Logo:** the current logo (navy circle + gold ring) predates this rebrand and now clashes with the new palette. Flagged as an open item below — don't recolor it ad hoc; it needs a real redesign pass.

### Visual style rules

- Clean, crisp, minimal — sharp corners (0–2px radius everywhere, no soft rounded cards), generous whitespace, no warm/earthy textures
- The accent blue is used sparingly — CTAs, links, small highlights — large surfaces stay white/charcoal/gray
- Motion/glow effects (e.g. the spotlight background) should use the accent blue at low opacity, not gold

- **2026-07-04 — full rebrand.** Previous identity (kept for reference only): navy `#1B2A3D` / burnt-gold `#CC6B49` / cream `#FBF1E6`, Playfair Display serif headings, soft 4–6px corner radii, warm/boutique/editorial style. Superseded by the clean/crisp/electric-blue system above.

## 3. Voice & tone

**Voice:** Confident, approachable, no-fluff. Talk to store owners like peers, not prospects. Never corporate-speak, never hard-sell desperation.

**Rules:**
- Short, punchy sentences over long compound ones
- Lead with the store owner's pain point before the pitch ("Your store looks amazing. So why isn't anyone buying?")
- Every piece reframes their problem as *visibility/strategy*, not *quality* — their product/store is never the thing implied to be bad
- No jargon without translation ("GA4 setup" always gets a plain-English payoff: "you finally know what's working")
- Self-aware humor is allowed in entertainment-pillar content; never in promotional/direct-offer content

**Never:**
- Don't disparage the store owner's existing work
- Don't use generic marketing clichés ("game-changer," "unlock your potential")
- Don't oversell with exclamation-heavy copy — confidence reads calm, not hyped (exception: high-energy trending-audio pairings on TikTok, where the *audio* carries the energy, not the on-screen text)

## 4. Content strategy framework

**Pillar mix (applies to any 7-piece content batch): 40% Educational · 30% Entertainment · 20% Inspirational · 10% Promotional**

| Pillar | Job | Example hook |
|---|---|---|
| Educational | Teach a pain point + reframe | "If you're still posting product photos with no strategy, this is your sign." |
| Entertainment | Pure relatability, no hard pitch in-video | "POV: you spent $0 on marketing and expected customers to just show up." |
| Inspirational | Brand story, direct-to-camera, warm | "You don't need a $5k/month agency to grow your Shopify store." |
| Promotional | Direct offer, clean and punchy | "Three packages. One goal: your store grows. Which one's you?" |

**CTA rotation — never repeat the same CTA twice in a row:**
`Link in bio` → `Comment a keyword` → `DM us` → `Book a free consult`

**Core hashtags (every post):** `#DharmasEmporium #ShopifyStore #EcommerceMarketing #ShopifyTips #SmallBusinessMarketing #ContentStrategy`

*Decided 2026-07-01: this documented set is canonical going forward. The actual Instagram calendar's Day 1/Day 2 posts used a different, undocumented hashtag set (`#shopifyowner #ecommerce #digitalmarketing #dharmasesthetic` etc.) — that was drift, not a deliberate variant. Left as-is historically since it's already-published content; every post from Day 3 onward should use the set above instead.*

**Topic-rotation hashtags:**
- Content/social: `#SocialMediaMarketing #TikTokMarketing #InstagramMarketing #ContentCreation`
- Store optimization: `#ShopifySEO #EcommerceGrowth #StoreOptimization #ConversionRateOptimization`
- AI/video: `#AIMarketing #AIVideo #VideoMarketing`

**Posting cadence:** TikTok 1x/day during launch weeks, settling to 3-4x/week. Instagram same pattern, settling to 4-5x/week. Test windows: 11am–1pm and 7–9pm.
